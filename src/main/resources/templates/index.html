Bem-vindo ao meu aplicativo Spring Boot com Thymeleaf e Bootstrap

Este é um exemplo de aplicativo web criado com Spring Boot, Thymeleaf e Bootstrap.

Neste aplicativo, você pode:

  • Visualizar informações e dados importantes.
  • Interagir com formulários e enviar dados para o servidor.
  • Exibir dados dinamicamente com Thymeleaf.
  • Utilizar estilos e layouts atraentes com Bootstrap.

Você pode experimentar e expandir este aplicativo para atender às suas necessidades.
